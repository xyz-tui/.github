# xyz-tui

> One terminal. Every package manager.

XYZ is a terminal-based universal dependency manager that aggregates pip, npm, and brew into one interactive TUI with AI-powered insights.

## What We Build

A terminal-based dependency manager that gives you a single interface across pip, npm, and brew — with fuzzy search, orphan detection, and AI-powered package explanations via Google Gemini.

Built at **BeaverHacks** (May 2026).
